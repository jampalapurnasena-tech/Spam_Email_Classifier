# Spam Email Classifier

This project is a Machine Learning based Spam Email Classifier developed using Python and Scikit-learn.

## Features

* Detects spam emails/messages
* Uses Naive Bayes Algorithm
* Text preprocessing
* Accuracy evaluation

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Dataset

SMS Spam Collection Dataset

## Output

The model predicts whether a message is Spam or Not Spam.
